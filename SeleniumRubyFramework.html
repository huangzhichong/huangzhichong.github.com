Getting Start

Setup Ruby/Selenium

Install Ruby

  • Ruby version 1.9.2-p290 is required.

  • Follow the intstruction here to have it installed

Install Selenium-WebDriver

  • once you have ruby installed, install the selenium-webdriver gem with,

      gem install selenium-webdriver

Install Rspec

  • We will use the Rspec as test framework

  • To install Rspec for ruby

      gem install rspec

Driver for browser

  • only Firefox driver is installed as default

  • Download the drivers

    • IE - download the driver from here

    • Chrome - download the driver from here

      more drivers

  • Install the drivers

    1. put the drivers into a folder, like c:\webdrivers or ~/webdrivers in unix-based system
